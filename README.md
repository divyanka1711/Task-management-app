# Team Task Management Web App

## 🚀 Live Demo
https://task-management-app-production-e747.up.railway.app

## 📌 Features
- User Authentication (Signup/Login)
- Project creation and team management
- Task assignment and tracking
- Role-based access (Admin / Member)
- Dashboard with task statistics

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- Deployment: Railway

## ⚙️ Setup Instructions

### 1. Clone repo
git clone https://github.com/divyanka1711/Task-management-app.git

### 2. Install dependencies

Backend:
cd task/server
npm install

Frontend:
cd client
npm install

### 3. Environment Variables

Create `.env` in server:

DATABASE_URL=your_database_url  
JWT_SECRET=your_secret  
PORT=5000  

### 4. Run locally

Backend:
npm start

Frontend:
npm run dev

## 🚀 Deployment

Deployed on Railway:

- Backend + Frontend served together
- PostgreSQL database integrated
- Environment variables configured in Railway
